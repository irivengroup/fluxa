# Maintenance du runtime avancé

## Objectif
Stabiliser la ligne V4.7.x autour du payload runtime, des hooks priorisés et du schéma enrichi.

## Points surveillés
- hooks à priorité identique
- mode swallow des hooks priorisés
- `metadataValue()` avec valeur par défaut
- présence ou absence du contexte runtime dans le schéma
